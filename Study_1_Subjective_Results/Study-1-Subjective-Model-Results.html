Study 1 — Subjective Ratings: Mixed Models

2025-04-25

Siyuan Huang

Overview

This document models overall response quality (the mean of the four rating dimensions) for Top vs. Bottom models, as judged by the two rater groups (students, instructors). The primary model (model1) clusters repeated ratings on the explanation output (Model × Task × Iteration) via (1 | Explanation) — the unit the eight raters score in common — together with (1 | Rater). This primary model is reported in full with Type-III ANOVA, effect sizes, post-hoc contrasts, and diagnostics. A supplementary section then refits the model with random effects of (1 | Attempt) together with (1 | Rater).

Recode Core Variables

Design variables are coerced to factors; a unique Rater identifier and the composite OverallQuality outcome are constructed, and models are grouped into Top/Bottom/Other from the objective benchmark ranking.

df <- df %>%
  mutate(
    Model = factor(Model),
    Profession = factor(Profession),
    Task = factor(Task),
    Iteration = factor(Iteration),
    ID = factor(ID),
    RaterGroup = Profession,
    Attempt = Iteration,
    Rater = factor(paste(RaterGroup, ID, sep = "_")),
    # Explanation = the unique output that all raters score in common
    # (Model × Task × Iteration), used as the repeated-measures
    # clustering unit in the primary model.
    Explanation = factor(paste(Model, Task, Iteration, sep = "_")),
    OverallQuality = rowMeans(
      across(
        c(
          Statistical_Accuracy,
          Clarity_and_Structure,
          Conceptual_Depth,
          Pedagogical_Effectiveness
        )
      ),
      na.rm = TRUE
    ),
    ModelGroup = case_when(
      str_detect(Model, regex("ChatGPT_4o", ignore_case = TRUE)) ~ "Top",
      str_detect(Model, regex("ChatGPT_o1", ignore_case = TRUE)) ~ "Top",
      str_detect(Model, regex("Grok3", ignore_case = TRUE)) ~ "Bottom",
      str_detect(Model, regex("Gemini_2\\.0", ignore_case = TRUE)) ~ "Bottom",
      TRUE ~ "Other"
    ),
    ModelGroup = factor(ModelGroup, levels = c("Bottom", "Top", "Other"))
  )

Model 1 — Primary: (1 | Rater) + (1 | Explanation)

# Model 1 (primary): repeated ratings are clustered on the
# explanation output (Model × Task × Iteration) via (1 | Explanation),
# the unit the 8 raters score in common.
model1 <- lmer(
  OverallQuality ~ ModelGroup * RaterGroup * Task +
    (1 | Rater) + (1 | Explanation),
  data = df
)
